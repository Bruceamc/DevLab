Estrutura completa

<!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document</title>
        </head>
        <body>
        </body>
        </html>

Podemos dividir esse documento em quatro partes principais:

  1. Declaração do documento (<!DOCTYPE html>).
  2. Elemento <html>.
  3. Elemento <head>.
  4. Elemento <body>.

<!DOCTYPE html>

O que é?

Essa linha informa ao navegador qual versão do HTML está sendo utilizada.

No HTML moderno, basta escrever:

<!DOCTYPE html>

Isso indica que o documento segue o padrão do HTML5.

Antes do HTML5

Antigamente a declaração era muito maior: 

<!DOCTYPE HTML PUBLIC
        "-//W3C//DTD HTML 4.01 Transitional//EN"
        "http://www.w3.org/TR/html4/loose.dtd">

Hoje isso não é mais necessário.

Por que é importante?

Sem essa declaração, alguns navegadores entram em um modo chamado Quirks Mode.

Nesse modo, eles tentam imitar o comportamento de navegadores muito antigos.

Isso pode causar problemas como:

  • CSS funcionando de maneira diferente;
  • Tamanhos incorretos;
  • Layouts quebrados.

Por isso, praticamente todo documento HTML moderno começa com: 

<!DOCTYPE html>

2. Elemento <html>

<html lang="en">

O que é?

Esse é o elemento raiz da página.

Todo o conteúdo do documento HTML fica dentro dele. 

<html>
            ...
        </html>

Sem esse elemento, o documento não estaria corretamente estruturado.

O atributo lang

O atributo lang define o idioma principal da página. 

lang="en"

Nesse caso:

  • en = English (Inglês).

Se a página estiver em português, o recomendado é utilizar: 

<html lang="pt-BR">

ou

<html lang="pt">

Por que isso é importante?

Definir corretamente o idioma da página traz diversos benefícios:

  • Melhora a otimização para mecanismos de busca (SEO);
  • Permite que leitores de tela utilizem a pronúncia correta;
  • Ajuda os navegadores a interpretar o conteúdo;
  • Facilita o funcionamento de tradutores automáticos.

Por exemplo, um leitor de tela altera automaticamente a pronúncia das palavras de acordo com o idioma definido em lang.

Estrutura geral

Dentro do elemento <html> existem duas grandes áreas: 

<html>
            <head>
            </head>
            <body>
            </body>
        </html>

3. Elemento <head>

<head>
        </head>

O que é?

O elemento <head> contém informações sobre a página.

Essas informações normalmente não aparecem visualmente para o usuário, mas são utilizadas pelo navegador, pelos mecanismos de busca e por outros serviços.

O que fica dentro do <head>?

  • Título da página;
  • Descrição (SEO);
  • Favicon;
  • Arquivos CSS;
  • Fontes;
  • Scripts;
  • Metadados.

Exemplo

<head>
            <title>Minha Página</title>
            <link rel="stylesheet" href="style.css">
            <script src="script.js"></script>
        </head>

4. <meta charset="UTF-8">

<meta charset="UTF-8">

O que é?

Define qual codificação de caracteres será utilizada pela página.

Atualmente, praticamente todos os sites utilizam o padrão UTF-8.

O que é UTF-8?

UTF-8 é um padrão internacional para representar caracteres.

Ele suporta:

  • Português;
  • Inglês;
  • Japonês;
  • Árabe;
  • Emojis 😀;
  • Praticamente todos os idiomas.

Sem UTF-8

O texto:

Olá

Poderia aparecer como:

OlÃ¡

ou

Ol?

6. Elemento <title>

<title>Document</title>

O que é?

Define o título da página.

Esse título aparece:

  • Na aba do navegador;
  • Nos favoritos;
  • Nos resultados dos mecanismos de busca.

Exemplo

<title>Curso de HTML</title>

A aba do navegador exibirá:

Curso de HTML

7. Elemento <body>

<body>
        </body>

O que é?

Todo o conteúdo visível para o usuário fica dentro do elemento <body>.

Exemplos de elementos

<h1>Título</h1>
        <p>Texto</p>
        <img>
        <button>
        <form>
        <nav>
        <footer>

Todos esses elementos pertencem ao <body>.

Exemplo

<body>
            <h1>Olá Mundo!</h1>
            <p>Minha primeira página.</p>
        </body>

Resultado

Olá Mundo!

Minha primeira página.

Hierarquia do documento

<!DOCTYPE html>
        └── html
            │
            ├── head
            │     ├── meta
            │     ├── meta
            │     └── title
            │
            └── body
                └── Conteúdo visível

Fluxo de leitura do navegador

Quando uma página é acessada, o navegador segue aproximadamente esta sequência:

  1. Lê <!DOCTYPE html> para identificar o padrão do documento.
  2. Encontra o elemento <html> e identifica o idioma definido pelo atributo lang.
  3. Processa o <head>, carregando metadados, estilos, fontes e scripts.
  4. Renderiza o conteúdo do <body>, exibindo a interface da página.
